// 为什么 JSONP 不受跨域限制？
//     因为 <script> 标签可以加载任何站点的 JS 文件，不受同源策略限制。
//     JSONP 本质上是“请求一段 JS 代码”，而不是 AJAX 请求。
// ⚠️ JSONP 的局限性
//         项目	        是否支持
//         GET 请求	    ✅ 支持
//         POST/PUT 等	    ❌ 不支持
//         错误处理	    ❌ 很难处理错误（除非超时）
//         安全性	        ⚠️ 有被注入攻击的风险还要这个
